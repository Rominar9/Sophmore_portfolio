Domain Summaries

Domain 1

This domain is about the creation and structuring of HTML documents, including embedding JavaScript into the code, linking external and internal resources, and defining metadata, fill in the blanks and creating script and Noscript tags.

Domain 2

In this domain I completed the steps provided, I learned how to create and apply various types of CSS styles, understand the procedures of style sheets, and use valid syntax for rule sets. Additionally, I applied IDs, classes, and pseudo code classes to HTML elements to control their appearance and behavior so that the user can easy navigate.

Domain 3

This domain is about comprehensive overview of Domain 3, it covers the fundamental HTML elements and attributes used to create and structure web content, enhancing both functionality and accessibility.

Domain 4

In this domain sections cover fundamental HTML skills for integrating media content which inlcude images, videos, audio and using HTML elements to control the layout and navigation of web content.

Domain 5

In this domain the course covers web design essentials like units of measure, responsive design techniques, frameworks, and CSS Grids. It teaches about measurements like em, ex, px, and percentages, along with viewport settings and media queries. Practical exercises reinforce these concepts. These are all essential when it comes to creating a good website.

Domain 6

In this domain each section includes a purpose statement outlining the learning objectives and steps for completion to reinforce the concepts discussed. Overall, the information aims to equip developers with essential skills and knowledge for creating well-structured, accessible, and visually appealing web content.
